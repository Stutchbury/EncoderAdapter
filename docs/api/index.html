EncoderAdapter

This library provides a very thin adapter for other encoder libraries, enabling them to all offer the same API.

Only three methods are specified for the adapter that must be implemented by concrete classes:

  • bool EncoderAdapter::begin(void)
  • int32_t EncoderAdapter::getPosition(void)
  • void EncoderAdapter::setPosition(int32_t)

Concrete implementations will likely have more methods and should implement those required to configure the underlying library.

To enable Arduino IDE to compile and link if no encoder is used, the adapter class must be 'header only'.

An adapter for PJRC's popular Encoder is provided as the PjrcEncoderAdapter class.

‍Note: Although this library lists no dependencies, it is dependent on whichever underlying encoder library is chosen.
